🚀 White-Label Chat Demo

This demo showcases the stealth chatbot widget with custom theming

Demo Site 1 - Blue Theme

This page demonstrates the white-label chat widget with a custom blue theme. The widget is completely stealth - no "Ritzie" branding visible to end users.

🥷 Stealth Features Demonstrated:

  • ✓ Shadow DOM encapsulation
  • ✓ Obfuscated CSS class names
  • ✓ Hashed CSS filenames
  • ✓ No vendor strings in bundle
  • ✓ Custom theme per site
  • ⚠ Vanity CDN (demo uses localhost)

🎨 Theme Configuration:

  • Primary Color: #007bff (Blue)
  • Font Family: Inter, sans-serif
  • Border Radius: 8px
  • Site Token: SITE_demo_site1

🔧 Technical Details:

  • Widget loads from: packages/embed/dist/chat.v1.js
  • Theme CSS: dist/themes/SITE_demo_site1/chat-ui.[hash].css
  • Analytics: Events sent to telemetry endpoint
  • Multi-tenant: Isolated by org_id + site_id
