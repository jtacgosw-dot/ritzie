Demo Site 2 - Green Theme

This demonstrates the same white-label widget with a completely different theme, showing multi-tenant theming capabilities.

🎨 Theme Configuration:

  • Primary Color: #28a745 (Green)
  • Secondary Color: #17a2b8 (Teal)
  • Font Family: Roboto, sans-serif
  • Border Radius: 12px
  • Site Token: SITE_demo_site2

🔒 Multi-Tenant Isolation:

  • Separate CSS bundle with different hash
  • Isolated analytics and conversation data
  • Different bot configuration and prompts
  • Scoped cache keys prevent cross-tenant leakage
